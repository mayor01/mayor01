## Featured Projects

### [Customer Churn Prediction](https://github.com/mayor01/Telecom-customer-churn-prediction)

Machine learning project predicting customer churn using Logistic Regression, KNN, Random Forest, and XGBoost. Includes SMOTE for class imbalance and hyperparameter tuning.

### [UK Regional Economic Analysis](https://github.com/mayor01/uk-regional-economic-analysis)

Interactive analysis of economic inactivity and deprivation indices across UK local authorities using ONS data. Features an HTML dashboard for data exploration.

### [Sales Analytics Dashboard](https://github.com/mayor01/sales-analytics-powerbi)

Power BI dashboard analyzing sales performance, customer behavior, and product trends across multiple branches.

### [Weather Data Analysis](https://github.com/mayor01/weather-data-analysis)

Python project for processing and visualizing weather data using APIs and SQLite3, focusing on temperature trends and rainfall analysis.

### [Lung Cancer Risk Prediction](https://github.com/mayor01/Lung-cancer-risk)

Predictive modeling project using Support Vector Machine (SVM) to assess lung cancer risk based on patient data.
